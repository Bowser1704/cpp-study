### 引用

引用是一个别名，它可以说完全是原变量，地址啊，值啊，都是，改变引用也会改变原来的值。

- 引用与指针的区别

  1. 不存在空引用。引用必须连接到一块合法的内存。
  2. 一旦引用被初始化为一个对象，就不能被指向到另一个对象。指针可以在任何时候指向到另一个对象。**即引用是不可以修改的**
  3. 引用必须声明时初始化,指针和普通变量一样.

  ```c++
  int i;				//变量
  int & a = i; 	//引用
  
  int & a;		//错误
  
  int* p;			//指针
  p = &a; 		
  
  int main(){
      int i = 1;
      int& a = i;
      int *p = &a;
      std::cout << i << '\n' << a << '\n' << p << '\n' << &i;
      return 0;
  }
  /*
  1
  1
  0x7ffc28b18a24
  0x7ffc28b18a24	地址是一样的
  */
  ```
